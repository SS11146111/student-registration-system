Student Registration System

A system to add, update & delete student records!!

The functionalities of this system include: adding, updating, deleting student record.

To add a student record, go to : Home-> fill the required details -> click - 'Add Record' button.
If the record is inserted successfully, a message, 'Student registered successfully!!!' will pop up.
The added record can be viewed in the 'View Records' table.

To deleted a record, go to : View Records -> click the corresponding delete button.

To edit a record, go to : View Records -> click the corresponding edit button.
A window will pop up where you can fill the updated information and then click 'Update Record' button.
The updated data will appear on the 'View Records' table.
